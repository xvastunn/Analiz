Розділ 7. Модель даних

Модель даних інформаційної системи бібліотеки включає такі основні сутності:

1. Книга (Book)

ID книги

Назва

Автор

Жанр

Рік видання

Статус (доступна / видана)

2. Користувач (User)

ID користувача

Ім'я

Прізвище

Тип користувача (адміністратор, бібліотекар, читач)

Контактні дані

3. Видача (Loan)

ID операції

ID книги

ID користувача

Дата видачі

Дата повернення

Статус

Зв’язки між сутностями:

Один користувач може мати багато видач.

Одна книга може брати участь у багатьох операціях видачі.

Видача пов’язує книгу та користувача.

Таким чином модель даних відображає основні бізнес-процеси бібліотеки та забезпечує коректну роботу системи.
