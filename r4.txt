Розділ 4. Функціональні сценарії (User Stories)
User Stories для адміністратора:

Як адміністратор, я хочу додавати нові книги в систему, щоб підтримувати актуальний каталог.

Як адміністратор, я хочу редагувати дані про книги, щоб виправляти помилки.

Як адміністратор, я хочу керувати обліковими записами користувачів, щоб контролювати доступ.

User Stories для бібліотекаря:

Як бібліотекар, я хочу реєструвати видачу книг, щоб вести облік літератури.

Як бібліотекар, я хочу фіксувати повернення книг, щоб оновлювати статус доступності.

Як бібліотекар, я хочу перевіряти наявність книги, щоб надавати інформацію читачам.

User Stories для читача:

Як читач, я хочу шукати книги за назвою або автором, щоб швидко знаходити потрібну літературу.

Як читач, я хочу переглядати доступність книги, щоб знати чи можу її отримати.

Як читач, я хочу переглядати історію своїх замовлень, щоб контролювати повернення.
